# 🔥 Python Firewall Simulator

A simple firewall simulation built in Python that demonstrates how basic IP filtering works.

This project generates random IP addresses and checks them against predefined firewall rules to either **Allow** or **Block** traffic.

---

## 📌 Features

- ✅ Random IP address generation
- ✅ IP rule-based blocking
- ✅ Simple firewall logic
- ✅ Clean and beginner-friendly Python code
- ✅ Easily extendable for advanced security concepts
